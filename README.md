## 📝 Descripción General del Sistema
Este sistema interactivo ejecutado por consola permite administrar de manera completa las reservas de entradas de un complejo cinematográfico para las películas: *Toy Story 5*, *Scary Movie 6* y *El Diablo Viste a la Moda 2*. 

El programa controla dinámicamente la capacidad de las salas por función, realiza el cálculo automatizado de los importes, aplica descuentos promocionales y genera estadísticas gerenciales detalladas (recaudación total acumulada y cálculo de la película con mayor demanda del día).
